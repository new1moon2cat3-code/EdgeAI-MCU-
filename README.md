## 🤖 二、期末專題：Edge AI 智慧助理

> 📡 Edge AI + IoT + 電腦視覺 + 語音辨識 + 感測器融合  
> 🎯 使用 AMB82-mini 建立完整智慧邊緣運算系統

🌟 專題概述
本專題整合 **AI + IoT + 嵌入式系統**，打造一個即時智慧助理系統。

### 🔧 核心功能

- 📷 Camera → AI 視覺辨識（GPT-4o）
- 🎤 麥克風 → 語音辨識（Whisper）
- 🧠 AI Server → 大語言模型處理
- 🔊 喇叭 → 語音合成（TTS）
- 🖥 TFT LCD → 即時顯示結果
- 🔘 按鈕 → 使用者互動控制


# 🧠 系統架構

```
                ┌──────────────┐
                │    按鈕      │
                └─────┬────────┘
                      ▼
        ┌──────────────────────────┐
        │      AMB82-mini         │
        │                          │
        │  📷 攝影機 → AI視覺      │
        │  🎤 麥克風 → 語音輸入    │
        │  📡 WiFi → AI伺服器     │
        │  🖥 TFT → 顯示結果      │
        │  🔊 喇叭 → 語音輸出      │
        └──────────┬──────────────┘
                   ▼
        ┌──────────────────────────┐
        │       AI 伺服器         │
        │                          │
        │  GPT-4o 視覺模型        │
        │  Whisper 語音辨識       │
        │  LLM 語意分析           │
        │  TTS 語音合成           │
        └──────────────────────────┘
* Grok
```

# 🧠 系統流程

```
          使用者按下按鈕
      ↓
拍攝影像 + 錄音
      ↓
上傳至 AI 伺服器
      ↓
AI 進行視覺 + 語音分析
      ↓
產生回覆結果
      ↓
TFT 顯示 + 喇叭播放

```
